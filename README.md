## Stack Tecnológico

- **Frontend:** React
- **Backend:** FastAPI (Python)
- **Base de datos:** MySQL
- **Despliegue:** Railway
- **Notificaciones:** Twilio

## Cómo correr el proyecto localmente

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
